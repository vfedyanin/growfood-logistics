## 18. Тарифы: скомпилированный тариф для группы компаний
Несколько юрлиц одной группы (например КВ + Перспектива) ездят в одном рейсе и делят стоимость.
Нужен механизм «компилированного тарифа» — один тариф на маршрут, стоимость делится между
участниками группы пропорционально паллетам. Связано с п.19 (группы компаний).

## 22. Счета: логика расчёта стоимости на основе типов тарифов
Текущая логика счётов не учитывает тип тарифа. Нужно реализовать расчёт суммы счёта:
- Тип A (за паллет): кол-во паллет × ставка
- Тип B (скомпилированный): рейсовая ставка ÷ паллет_всего × паллет_клиента
- Тип C (ступенчатый): выбор ставки по порогу паллет
- Тип D (ставка + фикс): паллеты × ставка + фикс
- Дополнительные услуги: суммируются сверху
Итог: система сама считает сумму счёта при его создании.
